overview flexbox is another css layout modal. flex is a block level element. flex consist of two element flex parent and flex childrens. a flex box element is also known an flex container, and their direct child known as flex items. flexbox is a single direction layout concept. flex items can only placed only vertically ( column ) or horizontally ( row ) at a time. Nesting of these boxes (horizontal inside vertical, or vertical inside horizontal) can be used to build layouts in two dimensions. flex is short name of flexible box module and flexbox. flex is very easy to learn and efficient to create responsive layouts. flexbox provide us control over the alignment, direction, order, and size of items. flex provide a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown and/or dynamic (thus the word "flex", why it is called flex or flexible box layout model ). The main idea behind the flex layout is to give the container the ability to alter its items' width/height (and order) to best fill the available space. A flex container expands items to fill available free space or shrinks them to prevent overflow. In the flex layout model, the children of a flex container can be laid out in any direction, and can “flex” their sizes, either growing to fill unused space or shrinking to avoid overflowing the parent. Both horizontal and vertical alignment of the children can be easily manipulated. important concept to understand when workinig with flex : main axis and cross axis start and end line start edge and end edge available space : when we put mulitple boxes in a row, there may be some free space available, Before we can make sense of these properties we need to consider the concept of available space. The main purpose of flex items are to let their container know how many things it needs to position. If the items don’t have a size then the content's size is used as the flex-basis. The flex-grow property can be used to distribute space in proportion.
